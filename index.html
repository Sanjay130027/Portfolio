Projects

AI Resume Classifier

Developed an AI-based resume classification system that automatically categorizes resumes into job roles such as Software Developer, Data Scientist, HR, and Marketing. Used the BERT Transformer model to understand the meaning of resume text and Support Vector Machine (SVM) for accurate classification. The project helps automate resume screening, improve classification accuracy, and reduce manual effort in the recruitment process.

AI Skill Identifier

Built an intelligent system that automatically detects and identifies candidate skills from resume data. The project uses a Convolutional Neural Network (CNN) to learn useful patterns from the resume text and a Random Forest model to classify the candidate's skills accurately. This solution makes the recruitment process faster by helping recruiters identify relevant skills with less manual effort.

Hybrid Resume Screening

Designed a hybrid AI system to improve the accuracy of resume screening by identifying and classifying candidate information effectively. The project combines Long Short-Term Memory (LSTM) for understanding resume text with Logistic Regression for classification. This approach helps recruiters filter resumes more accurately, reduces incorrect candidate selection, and supports a faster hiring process.

XGBoost Resume Analyzer

Created a machine learning system to analyze resumes and categorize them based on their information. The project compares the performance of XGBoost and K-Nearest Neighbor (KNN) to determine which algorithm produces more accurate results. XGBoost achieved better performance, making the system more reliable and efficient for resume analysis and candidate evaluation.
